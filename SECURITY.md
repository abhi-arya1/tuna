# Tuna Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Tuna, please report it to us ASAP. You can do so anonymously, or for-credit.

Please report it by sending an email to [abhi[at]opennote.me](mailto:abhi@opennote.me)
